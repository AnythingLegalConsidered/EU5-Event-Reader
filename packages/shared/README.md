# @eu5-reader/shared

Types, constantes et utilitaires partagés pour le monorepo EU5 Event Reader.

## Scripts

- `pnpm build` : génère les fichiers compilés dans `dist/`
- `pnpm type-check` : vérifie les types
- `pnpm clean` : supprime `dist/`

## Contenu

- `types/` : définitions TypeScript (événements, pays, localisation)
- `constants/` : constantes partagées
- `utils/` : utilitaires communs
